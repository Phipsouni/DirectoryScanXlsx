# DirectoryScan

Скрипт анализирует папки с PDF в указанной директории и формирует Excel-таблицу с номерами ЭСД и ГТД.

**Запуск:** `DirectoryScan.py` или `Start.bat`. При первом запуске зависимости установятся сами; пути к рабочей папке и папке сохранения запрашиваются и сохраняются в `path.txt`. Дальше открывается меню: запуск сканирования, смена путей. Пути можно вводить в кавычках или без.

**Логика данных:** название приложения берётся из имени папки (между 3-й и 4-й запятыми). ЭСД — PDF с 4 дефисами в названии; ГТД — файлы вида `GTD_<число>_<число>_<число>.pdf`. Папки без числа в начале имени не обрабатываются. Результат — `ESD_DT.xlsx` в выбранной директории сохранения.
